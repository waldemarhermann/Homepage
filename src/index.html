Face Detector

This is a web application that allows users to register and sign in, and then detect faces in images using Clarifai's API. Users can track their usage by uploading images and seeing the number of faces detected, as well as their total image upload count. The app uses Node.js and PostgreSQL on the backend, and is built with React on the frontend. It features user authentication, form validation, and database management using Knex.js.

Robocats

This is a project that displays a list of cats, which can be searched and filtered by name. The project makes use of a React component structure, with each component serving a specific purpose. The App component serves as the container for the overall application, and it manages the state of the list of cats and the search field value. The SearchBox component provides a search field that allows the user to filter the list of cats by name. The CardList component displays the list of cats, using a Card component for each individual cat.
